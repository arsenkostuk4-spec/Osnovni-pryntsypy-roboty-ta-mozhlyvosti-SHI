# Telegram Echo Bot (Ruby)

Простий Telegram-бот, створений на Ruby з використанням gem telegram-bot-ruby.
Бот повторює всі повідомлення користувача та реагує на команду /start.

## Встановлення та запуск
1. Встановити гем:
   gem install telegram-bot-ruby
2. Запустити бота:
   ruby bot.rb

## Команди
/start – стартове повідомлення  
Будь-який текст – бот повторює повідомлення
